Hello la team,

Little project using ReactJS, TypeScript, Tailwindcss, The PokeApi and more

## Getting Started

```bash
npm install
# or
yarn
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://127.0.0.1:5173/](http://127.0.0.1:5173/) with your browser to see the result.
